Interactive Tutorial

Code War is a strategic number battle game. Each player chooses a 4-digit secret code.

The goal is to correctly guess the opponent’s code.

  • Secret must contain 4 unique digits.
  • Guess CAN contain repeated digits.
  • Only numbers 0-9 allowed.

Dead = Correct number in correct position.

Injured = Correct number but wrong position.

First to get 4 Dead wins the round.

  • Easy — Random AI guesses
  • Normal — Smarter filtered guesses
  • Hard — Strategic AI algorithm

Local WiFi multiplayer will allow two devices on the same network to connect.

This feature is currently under development.
